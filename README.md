# Movie app

ReactJS fundamentals Course

# Requirements
1. Install nodeJS
2. Install Git
3. Install VSC
4. Open Terminal
5. 'npm install npx -g'

# SETUP
1. Open Terminal
2. Move to directory
3. 'npx create-react-app {directoryName}'

# Start
1. 'npm start'

# Create Git Repository
1. 'git init'
2. Create Repository in GitHub
3. 'git remote add origin {Repository URL}

# Update Git
1. git add .
2. git commit -m "{summary}"
3. git push origin master

# propTypes
